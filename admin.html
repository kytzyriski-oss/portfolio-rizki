⚙️ Setup Sekali Saja

Konfigurasi ini hanya perlu diisi satu kali. Setelah itu, klik 🚀 PUBLISH kapanpun kamu mau update website.

🐙
Step 1 — GitHub Token
Untuk auto-update data.json di repository kamu
Belum dikonfigurasi
Cara dapat GitHub Token (2 menit):
1. Buka github.com/settings/tokens/new
2. Isi Note: "Portfolio CMS"
3. Expiration: No expiration
4. Centang repo (full control) → klik Generate token
5. Copy token (ghp_xxxx...) → paste di bawah
6. Buat repository baru di GitHub bernama portfolio-rizki, upload semua file portfolio ke sana
🔗
Step 2 — Hubungkan Netlify ke GitHub
Agar Netlify auto-deploy setiap ada update di GitHub
Perlu dikonfigurasi
Lakukan ini sekali saja di Netlify:
1. Buka app.netlify.com → Login
2. Klik Add new site → Import an existing project
3. Pilih GitHub → authorize → pilih repo portfolio-rizki
4. Build settings biarkan default → klik Deploy site
5. ✅ Selesai! Setiap kamu klik Publish, Netlify otomatis update dalam ~30 detik
☁️
Step 3 — Cloudinary (Upload Foto)
Untuk upload foto profil & project ke cloud
Belum dikonfigurasi
Daftar & setup Cloudinary gratis:
1. Daftar di cloudinary.com
2. Dashboard → catat Cloud Name
3. Settings ⚙️ → Upload → Add upload preset → Signing mode: Unsigned → Save
4. Copy nama preset → paste di bawah
📊 Status Setup:
🎉 Semua sudah dikonfigurasi! Kamu bisa klik 🚀 PUBLISH kapanpun.
